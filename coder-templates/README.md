# Coder workspace templates

Terraform templates for Coder workspaces:

- **`docker-dev/`** — Docker-based workspace
- **`k8s-dev/`** — Kubernetes-based workspace

Project conventions and CI: root **`AGENTS.md`**. Pushing changes under **`coder-templates/`** to **`main`** runs **Deploy Coder Template (AWS)** (or run that workflow manually with **workflow_dispatch**).
